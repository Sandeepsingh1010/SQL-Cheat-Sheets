## 🎯 Learning Path

1. Start with `01_SELECT.md` - Learn basic data retrieval
2. Add filtering with `02_WHERE.md`
3. Combine conditions with `03_AND_OR_NOT.md`
4. Sort with `04_ORDER_BY.md`
5. Remove duplicates with `05_DISTINCT.md`
6. Limit results with `06_LIMIT_ROWNUM.md`
7. Aggregate with `07_GROUP_BY.md`
8. Filter groups with `08_HAVING.md`

## 📝 What You'll Learn

- Retrieving data from tables
- Filtering and sorting results
- Removing duplicate rows
- Grouping and aggregating data
- Combining multiple conditions
