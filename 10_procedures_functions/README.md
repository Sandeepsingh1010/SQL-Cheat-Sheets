# Procedures & Functions

Creating reusable PL/SQL procedures and functions.

## 📚 Topics Covered

| File | Description |
|------|-------------|
| [01_PROCEDURES.md](01_PROCEDURES.md) | Creating and calling stored procedures |
| [02_FUNCTIONS.md](02_FUNCTIONS.md) | Creating user-defined functions |

## 🎯 Learning Path

1. Master procedures with `01_PROCEDURES.md` - Reusable code blocks
2. Create functions with `02_FUNCTIONS.md` - Return values

## 📝 What You'll Learn

- Creating stored procedures
- Calling procedures with parameters
- IN, OUT, and IN OUT parameters
- Creating user-defined functions
- Returning values from functions
- Error handling and exceptions
- Dropping procedures and functions
- Package procedures and functions

## 💡 Key Differences

- **Procedure**: Does not return a value, performs actions
- **Function**: Returns exactly one value
- Both use parameters (IN, OUT, IN OUT)
- Both fully support PL/SQL logic
