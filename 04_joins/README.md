# Joins

Combining data from multiple tables using different join types.

## 📚 Topics Covered

| File | Description |
|------|-------------|
| [01_INNER_JOIN.md](01_INNER_JOIN.md) | Returns matching rows from both tables |
| [02_LEFT_JOIN.md](02_LEFT_JOIN.md) | All rows from left table + matching right rows |
| [03_RIGHT_JOIN.md](03_RIGHT_JOIN.md) | All rows from right table + matching left rows |
| [04_FULL_JOIN.md](04_FULL_JOIN.md) | All rows from both tables |
| [05_CROSS_JOIN.md](05_CROSS_JOIN.md) | Cartesian product - all combinations |

## 🎯 Learning Path

1. Start with `01_INNER_JOIN.md` - Learn basic table joining
2. Add unmatched rows with `02_LEFT_JOIN.md`
3. Learn `03_RIGHT_JOIN.md` and `04_FULL_JOIN.md`
4. Understand `05_CROSS_JOIN.md` for Cartesian products

## 📝 What You'll Learn

- Combining data from multiple tables
- Different join types and when to use them
- Join conditions and ON clause
- Handling unmatched rows (NULLs)
- Performance considerations
- Joining multiple tables

## 💡 Quick Reference

- **INNER JOIN**: Only matches
- **LEFT JOIN**: All from left + matches
- **RIGHT JOIN**: All from right + matches
- **FULL JOIN**: All from both
- **CROSS JOIN**: All combinations
