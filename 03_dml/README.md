# DML (Data Manipulation Language)

Inserting, updating, and deleting data in tables.

## 📚 Topics Covered

| File | Description |
|------|-------------|
| [01_INSERT.md](01_INSERT.md) | Adding new rows to tables |
| [02_UPDATE.md](02_UPDATE.md) | Modifying existing data |
| [03_DELETE.md](03_DELETE.md) | Removing rows from tables |

## 🎯 Learning Path

1. Start with `01_INSERT.md` - Learn to add data
2. Update data with `02_UPDATE.md`
3. Remove data with `03_DELETE.md`

## 📝 What You'll Learn

- Inserting single and multiple rows
- Using INSERT with SELECT
- Updating specific rows with conditions
- Deleting rows safely with WHERE clauses
- Best practices for data manipulation
- Transaction management (COMMIT/ROLLBACK)

## ⚠️ Important Notes

- Always use WHERE clause with UPDATE and DELETE
- Test DELETE with SELECT first
- Use transactions to ensure data consistency
