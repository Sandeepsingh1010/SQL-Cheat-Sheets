# DDL (Data Definition Language)

Creating, modifying, and deleting database objects.

## 📚 Topics Covered

| File | Description |
|------|-------------|
| [01_CREATE_TABLE.md](01_CREATE_TABLE.md) | Creating new tables with columns and data types |
| [02_ALTER_TABLE.md](02_ALTER_TABLE.md) | Modifying existing tables |
| [03_DROP_TABLE.md](03_DROP_TABLE.md) | Deleting tables and truncating data |
| [04_CONSTRAINTS.md](04_CONSTRAINTS.md) | Table constraints (PRIMARY KEY, FOREIGN KEY, etc.) |
| [05_LIST_TABLES.md](05_LIST_TABLES.md) | Querying the data dictionary to list tables |

## 🎯 Learning Path

1. Start with `01_CREATE_TABLE.md` - Learn to create tables
2. Add constraints with `04_CONSTRAINTS.md`
3. Modify with `02_ALTER_TABLE.md`
4. Clean up with `03_DROP_TABLE.md`
5. Query metadata with `05_LIST_TABLES.md`

## 📝 What You'll Learn

- Creating tables with proper data types
- Adding constraints (PRIMARY KEY, FOREIGN KEY, etc.)
- Modifying table structure
- Dropping and truncating tables
- Querying the data dictionary
- Maintaining database schema
