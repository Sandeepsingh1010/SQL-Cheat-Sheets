## 🎯 Learning Path

1. Understand structure with `01_BASIC_BLOCK.md`
2. Declare variables with `02_VARIABLES_DATATYPES.md`
3. Control flow with `04_CONDITIONS.md`
4. Iterate with `03_LOOPS.md`
5. Query rows with `05_CURSORS.md`
6. Handle errors with `06_EXCEPTIONS.md`
7. Continue with collections in `../16_collections/`

## 📝 What You'll Learn

- PL/SQL block structure (DECLARE, BEGIN, EXCEPTION, END)
- Variables and data type declarations
- IF/ELSIF/ELSE conditional logic
- FOR loops and WHILE loops
- Using cursors to fetch data
- Exception handling
- Procedural programming in Oracle

## 💡 Key Concepts

- **DECLARE**: Declare variables and cursors
- **BEGIN**: Main program logic
- **EXCEPTION**: Error handling
- **END**: Close the block
